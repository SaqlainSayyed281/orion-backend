# ORION Backend

Backend server for ORION AI Voice Assistant - Express API with PostgreSQL and Groq LLM integration.

## Tech Stack
- Node.js + Express
- PostgreSQL (Neon)
- Groq API (Whisper + Llama3 70B)
- JWT Authentication

## Setup
```bash
npm install
cp .env.example .env
# Edit .env with your credentials
npm run dev
